Setup Instructions
  1. In OBS, add a new Browser Source
  2. Click "Copy OBS URL" above and paste it as the URL
  3. Set width to 800 and height to 400 (adjust as needed)
  4. Enable "Refresh browser when scene becomes active"
  5. Click OK to save

Customize the overlay by adding URL parameters:
?layout=vertical&size=small&theme=light
